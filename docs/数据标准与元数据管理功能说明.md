# 数据标准与元数据管理系统功能说明

## 一、系统概述

### 1.1 系统定位
数据标准与元数据管理系统是数据治理平台的核心模块，负责建立和维护企业级数据标准，规范数据命名、格式和业务定义，同时管理元数据的采集、建模和查询功能。通过标准化的数据管理，提升数据质量，确保数据一致性和可用性。

### 1.2 核心功能模块
本系统包含以下主要功能模块：

| 模块名称 | 功能说明 | 主要页面 |
|---------|---------|---------|
| 数据标准管理 | 管理企业级数据标准，包括标准编号、命名规范、数据类型等 | standards/index.vue |
| 数据域管理 | 管理数据域定义，建立业务域与技术属性的映射 | domains/index.vue |
| 标准单词管理 | 管理标准化的中英文单词和缩写 | words/index.vue |
| 标准代码管理 | 管理枚举型数据的标准代码值 | codes/index.vue |
| 标准稽核 | 执行数据标准稽核任务，监控标准落地情况 | audit/index.vue |
| 元数据建模 | 构建元数据模型，定义实体绑定、血缘关系和标准映射 | metadata/modeling/index.vue |
| 元数据查询 | 提供多维度的元数据检索与查询服务 | metadata/query/index.vue |
| 元数据采集 | 管理数据抽取和同步任务 | metadata-collection/List.vue |

#### 2.1.2 功能特性

**列表字段说明**：

| 字段名称 | 数据类型 | 说明 | 必填 |
|---------|---------|------|------|
| 标准编号 | string | 标准的唯一标识符，格式如 STD-XXX-001 | 是 |
| 版本 | string | 标准版本号，格式为 v1.0 | 是 |
| 中文名称 | string | 标准的中文业务名称 | 是 |
| 英文简称 | string | 标准的英文缩写命名 | 是 |
| 主题 | string | 所属业务主题（如：客户主题、产品主题） | 否 |
| 域 | string | 所属数据域（如：客户域、交易域） | 否 |
| 数据类型 | string | 技术数据类型（String、Integer、Decimal等） | 是 |
| 归口管理部门 | string | 负责该标准的管理部门 | 否 |
| 状态 | string | 标准状态：已发布、草稿 | 是 |
| 更新时间 | string | 最后更新时间，格式：YYYY-MM-DD HH:mm:ss | 否 |

**操作按钮**：
- **查看**: 跳转至标准详情页，查看标准的完整信息
- **编辑**: 跳转至标准编辑页，修改标准信息
- **删除**: 弹出确认框，确认后删除标准

**搜索功能**：
- 支持按标准编号搜索（精确/模糊匹配）
- 支持按中文名称搜索
- 支持按英文简称搜索
- 搜索关键词不区分大小写

**批量导入**：
- 点击"批量导入"按钮，弹出导入弹窗
- 提供标准模板下载
- 支持 Excel 文件上传
- 根据标准编号自动识别更新或新增

- **数据属性卡片**: 展示数据的技术属性

  - 数据精度

  - 取值范围
  - 格式要求

- **业务属性卡片**: 展示数据的业务定义
  - 业务定义: 业务层面的数据含义和用途
  - 业务规则: 数据在业务处理中的规则约束
  - 来源依据: 数据的来源和依据说明

**关联资产 Tab**：
- 暂无关联资产时显示空状态

- 时间线展示标准的所有变更历史

- 示例：标准发布、专家审核通过、草稿创建等

**审批流 Tab**：
- 步骤条展示审批流程
- 包含：提交申请 → 数据专家审核 → 部门负责人终审 → 已发布
- 显示每个步骤的时间和状态

#### 2.2.3 操作按钮
- **返回列表**: 返回数据标准列表页
- **发布**: 仅在草稿状态下显示，点击发布标准
- **编辑**: 跳转到标准编辑页

### 7.2 实体绑定 Tab

#### 7.2.2 列表字段说明

| 字段名称 | 说明 |

#### 7.3.2 列表字段说明

  - 转换逻辑（文本框）
- **查看图谱**: 跳转到血缘图谱分析页面

### 7.4 标准映射 Tab

#### 7.4.1 功能说明
建立物理字段与数据标准之间的映射关系，支持智能匹配和手动修正。

#### 7.4.2 列表字段说明

| 字段名称 | 说明 |
|---------|------|
| 物理字段 | 数据库表中的字段名 |
| 所属表 | 字段所属的物理表 |
| 映射数据标准 | 字段映射到的数据标准（编号+名称） |

  - 所属表（只读显示）

  - 一致性状态（单选：符合/不符合）
