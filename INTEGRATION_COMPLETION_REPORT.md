### 🎯 技术架构

#### 集成后的目录结构
```
src/
├── pages/risk/model-offline-analysis/   # 模型离线分析模块
│   ├── Layout.vue                       # 统一布局组件
│   ├── featureCenter/                   # 特征中心
│   ├── modelRegister/                   # 模型注册
│   ├── modelBacktrack/                  # 模型回溯
│   ├── taskManagement/                  # 任务管理
│   └── modelEvaluation/                 # 模型评估
├── store/modules/model-offline.js       # 状态管理模块
├── router/model-offline-analysis.js     # 路由配置
└── config/menuConfig.js                 # 菜单配置
```
