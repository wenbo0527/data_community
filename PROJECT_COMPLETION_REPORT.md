## 🚀 快速开始

### 环境要求
- Node.js 18.0+
- npm 8.0+ 或 pnpm 8.0+

### 安装启动
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问应用
open http://localhost:5174/offline-model/demo
```
