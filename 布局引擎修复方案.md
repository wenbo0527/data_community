# 布局引擎修复方案 - 基于预览线系统统一接口

## 🎯 问题概述

用户报告PreviewLineSystem持续出现"布局引擎未就绪"错误，影响系统正常运行。经过深入调研发现了两个关键问题：

1. **函数未定义错误**: `TaskFlowCanvasRefactored.vue`中调用了未定义的`initializePreviewLineSystem`函数
2. **过度依赖布局引擎**: PreviewLineSystem强制要求布局引擎就绪才能创建预览线

## 🔍 问题分析

### 根本原因
- `initializePreviewLineSystem`函数在代码中被调用但从未定义，导致初始化失败
- PreviewLineSystem在`createUnifiedPreviewLine`方法中强制检查布局引擎状态，不满足条件时抛出错误
- 实际上PreviewLineSystem只在少数地方使用布局引擎，且都有备用方案

### 影响范围
- 预览线系统无法正常初始化
- 用户在控制台看到持续的错误日志
- 可能影响预览线的创建和显示功能

## 1. 问题分析总结

### 1.1 核心问题
- **初始化时序问题**: PreviewLineSystem在布局引擎就绪前初始化
- **耦合度过高**: 直接依赖布局引擎实例，缺乏解耦机制
- **错误处理不足**: 布局引擎未就绪时缺乏优雅降级

### 1.2 根本原因
```javascript

  // 问题：PreviewLineSystem在布局引擎创建前初始化
```

## 2. 解决方案设计

### 2.1 基于事件驱动的解耦架构

利用预览线系统的统一接口，实现事件驱动的初始化模式：

```javascript
// 新的解耦方案
class LayoutEngineManager {
  constructor() {
    this.layoutEngine = null;
    this.isReady = false;
    this.eventBus = new EventTarget();
    this.pendingOperations = [];
  }

  // 布局引擎就绪检查
  async waitForLayoutEngine(timeout = 5000) {
    if (this.isReady) return this.layoutEngine;
    
    return new Promise((resolve, reject) => {
      const timer = setTimeout(() => {
        reject(new Error('Layout engine initialization timeout'));
      }, timeout);

      const handler = () => {
        clearTimeout(timer);
        this.eventBus.removeEventListener('layout:ready', handler);
        resolve(this.layoutEngine);
      };

      this.eventBus.addEventListener('layout:ready', handler);
    });
  }

  // 设置布局引擎并触发就绪事件
  setLayoutEngine(engine) {
    this.layoutEngine = engine;
    this.isReady = true;
    
    // 触发就绪事件
    this.eventBus.dispatchEvent(new CustomEvent('layout:ready', {
      detail: { layoutEngine: engine }
    }));
    
    // 执行待处理操作
    this.processPendingOperations();
  }

  // 处理待处理操作
  processPendingOperations() {
    while (this.pendingOperations.length > 0) {
      const operation = this.pendingOperations.shift();
      try {
        operation(this.layoutEngine);
      } catch (error) {
        console.error('处理待处理操作失败:', error);
      }
    }
  }
}
```

```javascript
      this.isInitialized = true;
      console.log('✅ [PreviewLineSystem] 初始化完成');

      console.error('❌ [PreviewLineSystem] 初始化失败:', error);
      throw error;
```

## 3. 实现细节

### 3.1 布局引擎管理器

```javascript
// src/utils/layout/LayoutEngineManager.js
export class LayoutEngineManager {
  constructor() {
    this.layoutEngine = null;
    this.isReady = false;
    this.eventBus = new EventTarget();
    this.pendingOperations = [];
    this.readyPromise = null;
  }

  // 获取布局引擎（支持等待）
  async getLayoutEngine(timeout = 5000) {
    if (this.isReady) {
      return this.layoutEngine;
    }

    if (!this.readyPromise) {
      this.readyPromise = this.createReadyPromise(timeout);
    }

    return this.readyPromise;
  }

  // 创建就绪Promise
  createReadyPromise(timeout) {
    return new Promise((resolve, reject) => {
      if (this.isReady) {
        resolve(this.layoutEngine);
        return;
      }

      const timer = setTimeout(() => {
        reject(new Error(`Layout engine not ready within ${timeout}ms`));
      }, timeout);

      const handler = (event) => {
        clearTimeout(timer);
        this.eventBus.removeEventListener('layout:ready', handler);
        resolve(event.detail.layoutEngine);
      };

      this.eventBus.addEventListener('layout:ready', handler);
    });
  }

  // 设置布局引擎
  setLayoutEngine(engine) {
    if (this.isReady) {
      console.warn('Layout engine already set, ignoring duplicate call');
      return;
    }

    this.layoutEngine = engine;
    this.isReady = true;

    // 触发就绪事件
    const event = new CustomEvent('layout:ready', {
      detail: { layoutEngine: engine, timestamp: Date.now() }
    });
    this.eventBus.dispatchEvent(event);

    // 处理待处理操作
    this.processPendingOperations();

    console.log('✅ [LayoutEngineManager] 布局引擎设置完成');
  }

  // 添加待处理操作
  addPendingOperation(operation) {
    if (this.isReady) {
      // 立即执行
      try {
        operation(this.layoutEngine);
      } catch (error) {
        console.error('执行操作失败:', error);
      }
    } else {
      // 添加到待处理队列
      this.pendingOperations.push(operation);
    }
  }

  // 处理待处理操作
  processPendingOperations() {
    console.log(`🔄 [LayoutEngineManager] 处理 ${this.pendingOperations.length} 个待处理操作`);
    
    while (this.pendingOperations.length > 0) {
      const operation = this.pendingOperations.shift();
      try {
        operation(this.layoutEngine);
      } catch (error) {
        console.error('处理待处理操作失败:', error);
      }
    }
  }

  // 重置状态
  reset() {
    this.layoutEngine = null;
    this.isReady = false;
    this.pendingOperations = [];
    this.readyPromise = null;
  }
}
```
